# 🔐 Password Strength Analyzer

A Python-based tool that evaluates password strength using entropy, complexity checks, and prevents password reuse using a SQLite database.

## 🚀 Features
- Password strength evaluation (Weak / Moderate / Strong)
- Entropy calculation
- Complexity checks (uppercase, lowercase, digits, symbols)
- Common password detection
- Secure password storage using SHA-256 hashing
- SQLite database integration to prevent password reuse

## 🛠️ Technologies Used
- Python
- SQLite
- Hashlib (SHA-256)
- Regex

## ▶️ How to Run

```bash
python main.py
